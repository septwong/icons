# icons

## 使用

IconSet 是一个图标集合,可以从 GitHub Raw 下载图标文件

供 Quantumult X 使用，图标大小：108x108

```html
<!-- raw -->
https://raw.githubusercontent.com/septwong/icons/main/IconSet/OpenAI.png

<!-- CDN jsdelivr -->
https://cdn.jsdelivr.net/gh/septwong/icons@main/IconSet/OpenAI.png

<!-- GHProxy -->
https://ghproxy.com/https://raw.githubusercontent.com/septwong/icons/main/IconSet/OpenAI.png
```
